# Agent 01 — ROI Analysis (illustrative; customer-tunable)

> Numbers are a **worksheet**, not a guarantee. Plug in the customer's call volume,
> loaded labor rate, and deflection target. Comparable public results: Virginia
> Beach's AI assistant handled **1,300+ questions in month one** across 32 departments;
> Grupo TX's TxGov agent reported **~35% call-center volume reduction** and **~50%
> faster responses** (AWS, 2026).

## Value levers
1. **Call & email deflection.** Each contained self-service interaction avoids a live contact.
2. **Lower average handling time (AHT).** Grounded answers + correct routing reduce transfers.
3. **24/7 multilingual access.** Off-hours demand served without overtime.
4. **Fewer wrong-department transfers.** Intent + agency routing.
